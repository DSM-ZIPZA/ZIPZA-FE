# ZIPZA 부동산 플랫폼 - 디자인 아이디어

## 프로젝트 개요
지도와 매물 목록이 함께 표시되는 부동산 검색 플랫폼. 사용자가 지역의 매물을 지도상에서 시각적으로 탐색하고 상세 정보를 확인할 수 있습니다.

---

## 선택된 디자인 철학: **Modern Minimalism with Functional Elegance**

### 핵심 원칙
1. **지도 중심 인터페이스**: 지도가 주요 콘텐츠이며, 매물 목록은 보조 정보로 제공
2. **명확한 정보 계층**: 중요한 정보(가격, 위치, 면적)는 즉시 인식 가능하게 배치
3. **부드러운 상호작용**: 모든 전환과 애니메이션은 자연스럽고 의도적
4. **공간 활용**: 넉넉한 여백과 명확한 구분으로 시각적 호흡감 제공

### 색상 철학
- **Primary**: 신뢰와 안정성을 나타내는 깊은 파란색 (`#1e40af`)
- **Secondary**: 따뜻하고 친근한 오렌지 (`#f97316`) - 매물 강조
- **Background**: 깨끗한 흰색 (`#ffffff`) - 지도와의 대비
- **Text**: 어두운 슬레이트 (`#1e293b`) - 높은 가독성
- **Accent**: 부드러운 회색 (`#f1f5f9`) - 구분선과 배경

### 레이아웃 패러다임
- **분할 화면**: 좌측 매물 목록 (고정 너비 320px), 우측 지도 (유동적)
- **모바일**: 탭 기반 전환 (지도 ↔ 목록)
- **카드 기반**: 각 매물은 명확한 카드로 표현

### 시그니처 요소
1. **매물 카드**: 이미지, 가격, 위치, 면적이 한눈에 보이는 컴팩트 디자인
2. **지도 마커**: 가격 배지가 있는 커스텀 마커
3. **필터 바**: 상단에 고정된 필터 (지역, 가격대, 면적)

### 상호작용 철학
- 매물 카드 호버: 배경색 변화 + 지도 마커 강조
- 마커 클릭: 해당 매물 카드 자동 스크롤 + 상세 정보 표시
- 필터 변경: 부드러운 페이드 애니메이션으로 목록 업데이트

### 애니메이션 가이드
- **진입**: 카드는 아래에서 위로 슬라이드 (200ms, ease-out)
- **호버**: 카드 배경색 변화 (150ms, ease-in-out)
- **전환**: 지도 ↔ 목록 탭 전환은 페이드 (200ms)
- **로딩**: 부드러운 스켈레톤 로더 (pulse 애니메이션)

### 타이포그래피 시스템
- **Display**: Geist (700 weight) - 페이지 제목, 가격 표시
- **Body**: Inter (400, 500 weight) - 본문, 설명
- **Mono**: Fira Code - 가격, 면적 등 수치 정보
